## How to use

### Clone this repository

```sh
git clone https://github.com/linonon/obsidian_config.git
```

### Create a symbolic link

```sh
ln -fsn ./obsidian-vault/ {path_to_your_config}/.obsidian
```

### Open Obsidian

1. Select `Open folder as vault`
2. enjoy it!
